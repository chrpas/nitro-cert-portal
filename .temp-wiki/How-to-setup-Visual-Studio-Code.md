If you are going to edit TypeScript files you need to use Visual Studio Code to automatically format your code and fix linting issues. 

1. Open the project root directory with VS Code
2. Install "ESLint" extension
3. Install "Prettier - Code formatter" extension
4. Press CTRL+P type `>TypeScript` select "TypeScript: Select TypeScript Version", then select "Use Workspace Version" (without this you'll get errors and warnings if your TypeScript version differs from the project version)
