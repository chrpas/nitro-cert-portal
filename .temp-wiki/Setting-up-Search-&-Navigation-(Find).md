## Indexes in DXP
Indexes for the different environments in DXP are provisioned automatically by Optimizely. However, you need to specify (by emailing support@optimizely.com) which languages you want the indexes enabled for. Default enabled are Swedish and English.

## Getting started
Run the standard Optimizely job "Search & Navigation Content Indexing Job" once initially (you'll find it in backoffice CMS/Admin/Scheduled Jobs). After that, you only need the Nitro "Find indexing" jobs.
